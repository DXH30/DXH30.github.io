Aplikasi Android yang berfaedah

# Termux

Termux merupakan aplikasi virtual linux environment dalam android yang dapat digunakan untuk menjalankan program linux, maupun melakukan pemrograman.
Fitur :
1. Tanpa root
2. Untuk root virtual bisa pakai proot
3. Bisa install Vim atau Emacs
4. pkg = apt, if you know what I mean
5. ssh server (portnya dikasih 8022)
6. Direct access to com.* untuk menjalankan service Android seperti FrontCAM dll
7. Di close masih jalan service nya (biasanya)

Penting untuk install ini, kalau mau hacking, atau CTF :
1. Metasploit Framework (Exploit)
2. SQLMap (Buat SQLi)
3. Redare2 (Buat Reverse Engineering)
4. GDB (Buat Debugging)
5. Gnu-ToolChain (isinya compiler dan lain-lain)
6. OpenSSH (Install dulu kalau belum)
7. Compiler, Interpreter, buat main proglang mu. (Ex: python, perl, gcc, javac, ghci)

# X-Plore

Merupakan File Manager yang sudah ada sejak Jaman Symbian, yang lumayan terkenal bagi Developer dan Programmer saat itu. Merupakan software yang dibuat oleh **Lonely cat Games**. Fiturnya :
1. SSH/SFTP Client
2. FTP Client
3. Cloud Storage Client
4. WiFi Sharing Server (Via HTTP)
5. SMB Client (LAN Sharing Server)
6. DLNA
7. Vault (Sembunyikan file)
8. Terminal Emulator (Untuk SSH/SFTP atau HP yang di root)
9. Remount FileSystem as Root
10. Hexeditor
11. BuiltIn TextEditor
12. BuiltIn PDF-Reader
13. BuiltIn Music-Player
14. BuiltIn Video-Player
15. BuiltIn Picture-Viewer
16. MetaData Viewer
17. MP3 Tag Edit
18. NO ADS
19. DONATE untuk dapat fitur yang banyak
20. Jauh Lebih Ringan daripada FileManager yang lain
21. Sudah lebih dulu ada ketimbang File Manager yang lain (Zaman Symbian, sudah ada Hexeditor nya)
22. Worth for Real Programmer
