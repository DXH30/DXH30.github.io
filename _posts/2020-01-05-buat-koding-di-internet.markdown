Mau koding sekali bisa dibuka dimana-mana ? di tes dimana-mana ? Sebenarnya sih pilihannya, tapi untuk sementara ini saya coba pakai gitpod, untuk review nya mungkin akhir bulan januari ini akan saya review hasil review jujur.
Buat apa pakai online kalau kita sudah pakai yang offline ? Ya kebetulan saat ini era dimana internet itu bukan hal yang jarang, tapi juga kalau pakai SSD size memori makin kecil, tapi kencang untuk harga yang setara dengan HDD size besar.
Jadi faktor yang mendasari menggunakan IDE online adalah : 
 1. DevOps (Hampir seluruh service dikontrol lewat Internet)
 2. Internet makin murah dan makin umum ditemukan
 3. Harddisk dan Memori offline dapat mengancam keselamatan file kita
 4. Webbased application sudah mendominasi (vscode itu juga dibuat pakai JavaScript NodeJS yakni dengan framework ElectronJS)
Mungkin itu saja yang mendasari saya menggunakan IDE online seperti GitPod. Oh iya, sekedar informasi. Ekstensi Gitpod itu caranya dipakai langsung buka lewat situs github atau gitlab dan coba buka salah satu reponya, nanti akan muncul tombol Gitpod disitu, itu di klik.
