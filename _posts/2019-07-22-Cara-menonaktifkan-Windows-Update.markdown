Windows 10 suka update sendiri, cara mematikannya bisa :

1. Buka Start

2. ketik "gpedit.msc" lalu klik OK
<center><img src="/assets/img/2019/07/22/1.png"></center>
	   
3. Cari Computer Configuration\Administrative Templates\Windows Components\Windows Update
<center><img src="/assets/img/2019/07/22/2.png"></center>

4. Dobel klik Configure Automatic Updates

<center><img src="/assets/img/2019/07/22/3.png"></center>
5. Check Disable

6. Klik Apply

7. Klik OK
	
Setelah itu Windows akan berhenti update
