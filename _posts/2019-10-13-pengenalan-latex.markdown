LaTeX, dimana X nya dibaca ch dari huruf X (Chi) abjad yunani, merupakan cara lain mengetik dokumen, dan merupakan teknik yang lebih dahulu ada dari word processing. LaTeX ditulis sebagaimana menulis sebuah program. LaTeX dapat dikatakan bahasa pemrograman juga, mungkin lebih tepatnya MarkUp Language yang memiliki beberapa hal yang biasa ada di Bahasa Pemrograman seperti  pendefinisian fungsi dan kondisi dan class serta package.

Beberapa aplikasi offline untuk menggunakan latex adalah :
 - Emacs AucTeX (Integrasi dengan editor sehari-hari yaitu emacs)
 - TeXStudio (Dapat dipakai di semua sistem operasi)

Dan untuk online nya saya biasa menggunakan :
 - Overleaf (Khusus untuk pengguna latex murni) untuk kolaborasi dan git repo support harus bayar
 - Authorea (Digabung dengan Rich Text Format seperti word) cocok untuk kolaborasi dengan non pengguna latex
